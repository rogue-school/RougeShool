# TMP 폰트 생성기 (한글 완벽 지원)

TextMesh Pro 폰트를 **영어, 한글, 특수문자 완벽 지원**으로 생성/수정하는 Unity 에디터 툴입니다.

## 특징

### ✨ 한글 완벽 지원
- 한글 완성형 전체 (가-힣, 11,172자) 지원
- **Dynamic Mode** 자동 설정으로 런타임에 필요한 글자 자동 추가
- 한글이 빠지는 문제 완전 해결

### 🚀 간단한 사용법
- 직관적인 4단계 UI
- 원클릭 폰트 생성/수정
- 프로젝트 전체 폰트 일괄 교체

### ⚡ 최적화
- 4096x4096 Atlas 권장 (한글용)
- Dynamic Mode로 메모리 효율적
- 자동 Atlas 크기 계산

## 사용 방법

### 1. 툴 실행

Unity 상단 메뉴에서:
```
Tools > TMP > Font Generator (한글 지원)
```

### 2. 새 폰트 생성하기

#### Step 1: 폰트 선택
- **소스 폰트** 필드에 `.otf` 또는 `.ttf` 폰트 파일 드래그

#### Step 2: Atlas 설정
- **4096 ★** 버튼 클릭 (한글 지원용 권장)
- Point Size: 60 (기본값, 조절 가능)

#### Step 3: 포함할 문자 선택
- ✅ ASCII (영어, 숫자, 기본 기호)
- ✅ 한글 완성형 (가-힣, 11,172자)
- ✅ 특수문자 (화살표, 도형 등)

#### Step 4: 생성
- **"새 TMP 폰트 생성하기"** 버튼 클릭
- 저장 위치 선택
- 완료!

### 3. 기존 폰트 수정하기

1. **수정할 TMP 폰트** 필드에 기존 TMP Font Asset 선택
2. Atlas 크기 조정 (4096 권장)
3. **"선택한 폰트 에셋 수정하기"** 버튼 클릭

### 4. 프로젝트 전체 폰트 교체

1. 교체할 폰트를 **수정할 TMP 폰트** 필드에 선택
2. **"프로젝트 전체 폰트 교체"** 버튼 클릭
3. 모든 씬과 프리팹의 TMP 폰트가 자동 교체됩니다

## 권장 설정

### 🎮 한글 게임 개발용 (권장)

```
Atlas 크기: 4096x4096
Point Size: 60
Padding: 5
Mode: Dynamic (자동)

포함 문자:
✅ ASCII
✅ 한글 완성형
✅ 특수문자
```

**결과:**
- 예상 문자 수: 약 11,300자
- 런타임에 필요한 글자 자동 추가
- 한글 빠짐 현상 없음

### 🌍 영어 전용 게임

```
Atlas 크기: 2048x2048
Point Size: 90
Padding: 5
Mode: Dynamic (자동)

포함 문자:
✅ ASCII
✅ 특수문자
❌ 한글 완성형
```

## Dynamic Mode란?

**Dynamic Mode**는 TMP의 특별한 기능으로:

- ✅ **런타임에 필요한 글자를 자동으로 Atlas에 추가**
- ✅ 메모리 효율적 (사용하지 않는 글자는 로드 안 함)
- ✅ 한글처럼 문자가 많아도 문제없음
- ✅ Atlas 크기만 충분하면 모든 글자 표시 가능

이 툴은 **모든 폰트를 자동으로 Dynamic Mode로 생성**하므로, 한글이 빠지는 문제가 발생하지 않습니다.

## 문제 해결

### Q: 한글이 여전히 일부 표시되지 않아요

**A:** Atlas 크기를 확인하세요.

1. 폰트 에셋 선택
2. Inspector에서 `Atlas Width/Height` 확인
3. 2048x2048 이하라면 → 4096x4096으로 증가
4. 이 툴로 폰트 수정하여 Atlas 크기 증가

### Q: 폰트가 흐릿해요

**A:** Render Mode 문제입니다.

- 이 툴은 자동으로 **SDFAA** 모드 사용 (깔끔한 SDF)
- 기존 폰트라면 다시 생성 권장

### Q: 생성 후 글자가 안 보여요

**A:** Dynamic Mode는 런타임에 글자를 추가합니다.

- Unity 에디터에서는 비어 보일 수 있음
- **게임을 실행하면** 자동으로 글자가 표시됨
- 정상 동작입니다!

### Q: 프로젝트 전체 교체가 안 돼요

**A:** 다음을 확인하세요.

1. 씬이 저장되었는지 확인
2. 교체할 폰트가 올바르게 선택되었는지 확인
3. 콘솔에 에러 메시지가 있는지 확인

## 기술 정보

### 지원 문자 범위

| 종류 | 범위 | 문자 수 |
|------|------|---------|
| ASCII | U+0020 ~ U+007E | 95자 |
| 한글 | U+AC00 ~ U+D7A3 | 11,172자 |
| 특수문자 | 다양 | 약 50자 |

### Atlas 크기별 권장 용도

| Atlas 크기 | Point Size | 권장 용도 |
|-----------|-----------|----------|
| 2048x2048 | 90 | 영어 전용 게임 |
| 4096x4096 | 60 | **한글 게임 (권장)** |
| 8192x8192 | 50 | 대용량, 모든 문자 |

### Dynamic Mode의 작동 원리

1. 게임 실행 시 빈 Atlas로 시작
2. TMP가 텍스트를 렌더링할 때 필요한 글자 감지
3. Atlas에 해당 글자가 없으면 자동 추가
4. Atlas에 추가된 글자는 캐시되어 재사용
