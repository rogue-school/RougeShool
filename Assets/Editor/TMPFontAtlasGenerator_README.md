# TMP Font Atlas Generator

TextMesh Pro 폰트를 영어, 한글, 특수문자 완벽 지원으로 생성/수정하는 Unity 에디터 툴입니다.

## 주요 기능

### 1. 완벽한 한글 지원
- **한글 완성형 전체** (가-힣, 11,172자) 지원
- 자동 최적화된 Atlas 크기 추천
- Dynamic Mode로 메모리 효율적 운영

### 2. 다양한 문자 지원
- **ASCII**: 영어, 숫자, 기본 기호
- **한글**: 가-힣 전체
- **특수문자**: 게임에서 자주 사용하는 화살표, 도형, 원형 숫자 등
- **커스텀 문자**: 원하는 문자를 직접 추가 가능

### 3. 최적화 옵션
- **Dynamic Mode**: 런타임에 필요한 글자 자동 추가 (권장)
- **Static Mode**: 미리 지정된 문자만 포함
- **Atlas 크기 자동 계산**: 선택한 문자에 맞는 최적 크기 추천

### 4. 일괄 폰트 교체
- 프로젝트의 모든 씬과 프리팹에서 폰트 일괄 교체
- 안전한 백업 및 복구 프로세스

## 사용 방법

### 메뉴에서 실행
```
Unity 상단 메뉴 > Tools > TMP > Font Atlas Generator
```

### 새 폰트 생성하기

1. **소스 폰트 파일 선택**
   - `.otf` 또는 `.ttf` 폰트 파일을 선택합니다
   - 예: `SUIT-Bold.otf`, `Danjo-bold-Regular.otf`

2. **Atlas 설정**
   - **Atlas 크기**: 한글 지원을 위해 4096x4096 이상 권장
   - **Point Size**: 30~120 (작을수록 더 많은 글자 수용 가능)
   - **Render Mode**: SDFAA (권장) - 깔끔한 SDF 렌더링

3. **포함할 문자 선택**
   - ✅ ASCII 문자 (영어, 숫자, 기본 기호)
   - ✅ 한글 완성형 (가-힣, 11,172자)
   - ✅ 게임용 특수문자 (화살표, 도형 등)
   - 추가 커스텀 문자 입력 가능

4. **최적화 옵션**
   - ✅ **Dynamic Mode 사용** (권장)
     - 런타임에 필요한 글자 자동 추가
     - 메모리 효율적
     - 큰 Atlas 필요 (4096x4096 이상)

5. **생성하기**
   - "새 TMP 폰트 에셋 생성하기" 버튼 클릭
   - 저장 위치 선택
   - 완료!

### 기존 폰트 수정하기

1. **기존 TMP 폰트 에셋 선택**
   - 수정할 TMP Font Asset 선택
   - 자동으로 현재 설정 로드

2. **설정 변경**
   - Atlas 크기, Point Size 등 원하는 대로 조정
   - Dynamic Mode 활성화/비활성화

3. **수정하기**
   - "기존 폰트 에셋 수정하기" 버튼 클릭
   - 주의: 이 작업은 되돌릴 수 없습니다

### 프로젝트 전체 폰트 교체

1. **교체할 폰트 선택**
   - 기존 TMP 폰트 에셋 필드에 원하는 폰트 선택

2. **일괄 교체 실행**
   - "모든 씬/프리팹의 폰트 일괄 교체" 버튼 클릭
   - 확인 대화상자에서 "교체" 클릭

3. **결과 확인**
   - 씬과 프리팹에서 변경된 컴포넌트 수 표시

## 권장 설정

### 한글 게임 개발용 (권장)

```
Atlas 크기: 4096x4096
Point Size: 60
Padding: 5
Render Mode: SDFAA
Dynamic Mode: ✅ 활성화

포함 문자:
✅ ASCII 문자
✅ 한글 완성형
✅ 게임용 특수문자
```

**예상 문자 수**: 약 11,300자
**Atlas 사용률**: 약 70% (적정)

### 영어 전용 게임

```
Atlas 크기: 2048x2048
Point Size: 90
Padding: 5
Render Mode: SDFAA
Dynamic Mode: ✅ 활성화

포함 문자:
✅ ASCII 문자
✅ 게임용 특수문자
```

**예상 문자 수**: 약 300자
**Atlas 사용률**: 약 10% (여유)

### 모든 문자 지원 (최대)

```
Atlas 크기: 8192x8192
Point Size: 50
Padding: 3
Render Mode: SDFAA
Dynamic Mode: ✅ 활성화

포함 문자:
✅ ASCII 문자
✅ 한글 완성형
✅ 게임용 특수문자
+ 추가 커스텀 문자
```

**예상 문자 수**: 약 11,500자
**Atlas 사용률**: 약 40% (여유)

## 문제 해결

### Q: 한글이 일부 표시되지 않아요

**A**: 다음을 확인하세요:

1. **Atlas 크기가 충분한가?**
   - 4096x4096 이상 권장
   - 8192x8192로 증가 시도

2. **Dynamic Mode가 활성화되어 있는가?**
   - Dynamic Mode 체크박스 확인
   - Static Mode는 미리 정의된 문자만 표시

3. **Atlas가 꽉 찼는가?**
   - Point Size를 줄이기 (60 → 50)
   - Padding을 줄이기 (5 → 3)

### Q: 폰트가 뿌옇게 보여요

**A**: Render Mode를 확인하세요:
- **SDFAA** 또는 **SDF** 사용 권장
- SMOOTH, RASTER 모드는 확대 시 픽셀화됨

### Q: 메모리 사용량이 너무 높아요

**A**: 최적화 방법:
1. **Dynamic Mode** 활성화
2. Atlas 크기 줄이기 (8192 → 4096)
3. Point Size 줄이기
4. 사용하지 않는 문자 제외

### Q: 생성된 폰트 에셋이 비어있어요

**A**: Unity 에디터에서 추가 작업 필요:
1. 생성된 Font Asset 선택
2. Inspector에서 "Font Asset Creator" 열기
3. "Generate Font Atlas" 버튼 클릭
4. 또는 Dynamic Mode 활성화 후 게임 실행

## 기술 사양

### 지원 문자 범위

- **ASCII**: U+0020 ~ U+007E (95자)
- **한글**: U+AC00 ~ U+D7A3 (11,172자)
- **특수문자**: 약 200자
  - 도형: ■□◆◇★☆
  - 화살표: →←↑↓↔
  - 원형 숫자: ①②③④⑤
  - 괄호형: ⓐⓑⓒⓓⓔ
  - 로마 숫자: ⅠⅡⅢⅣⅤ
  - 그리스 문자: αβγδε

### Atlas 크기별 수용 가능 문자 수

| Atlas 크기 | Point Size | 예상 수용 문자 수 |
|-----------|-----------|----------------|
| 2048x2048 | 60        | ~2,800자       |
| 2048x2048 | 90        | ~1,200자       |
| 4096x4096 | 60        | ~11,200자      |
| 4096x4096 | 90        | ~4,800자       |
| 8192x8192 | 60        | ~44,800자      |
| 8192x8192 | 90        | ~19,200자      |

※ 실제 수용량은 Padding, Render Mode에 따라 달라질 수 있습니다.
